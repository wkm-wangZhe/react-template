#### 前言 
本项目仅供学习，新项目建议使用vue-template项目


#### 起步
 - 安装 
    
    `
        npm install 
    `
 -  运行 

    `
        npm run dev
    `
 -  打包:测试

    `
        npm run build:test
    `
 -  打包:生产

    `
        npm run build:PRO_GH
    `
